pantable.read_csv module¶

pantable.read_csv.parse_alignment(alignment_string, n_col)[source]¶

alignment string is parsed into pandoc format (AlignDefault, etc.). Cases are checked:

  • if not given, return None (let panflute handle it)

  • if wrong type

  • if too long

  • if invalid characters are given

  • if too short

pantable.read_csv.read_csv(include, data, encoding=None, csv_kwargs={})[source]¶

Parse CSV table.

include: path to CSV file or None. This is prioritized first.

data: str of CSV table.

Return None when the include path is invalid.

pantable.read_csv.regularize_table_list(raw_table_list)[source]¶

When the length of rows are uneven, make it as long as the longest row.

raw_table_list modified inplace.
